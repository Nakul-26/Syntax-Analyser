Control Statement Grammar
=========================

Supported program forms:
- statement_list
- int main ( ) block

Supported declaration:
- int variable ;
- float variable ;

Supported condition:
- variable relop variable
- variable relop number

Supported assignment:
- variable = expression ;

Supported expression:
- expr -> term ((+ | -) term)*
- term -> factor ((* | /) factor)*
- factor -> variable | number

Supported increment:
- variable ++ ;

Supported blocks:
- { statement_list }

Supported statement_list:
- statement statement_list
- epsilon

Supported statements:
- declaration
- assignment
- increment
- if ( condition ) statement else_part
- while ( condition ) statement
- block

Supported else_part:
- else statement
- epsilon

AST notes:
- declarations store type and identifier as child nodes
- assignments store target and expression as child nodes
- binary operators become expression nodes such as +, -, *, /
- conditions store the relational operator as the node value
